## VM Docker Containers

### DeerFlow (Super-Agent)

The main AI agent system using LangGraph for workflow orchestration.

| Sub-Component | Port | Image | Status | Role |
|---------------|------|-------|--------|------|
| **nginx** | 2026 | nginx | Active | Reverse proxy, main entry point |
| **LangGraph Engine** | 2024 | — | Active | Agent engine, 10 workers |
| **Frontend** | 3100 | Next.js | Active | Web UI |
| **Gateway** | 8001 | — | **⚠️ Config Issue** | API gateway |

**Capabilities:** Browser automation, shell access, filesystem access via LangGraph tools.

**Dependencies:** LM Studio (inference), Vault (knowledge base)

**Health Check:**
```bash
curl -sf http://localhost:2026
curl -sf http://localhost:2024/health
```

**Known Issue:** Gateway (port 8001) has a config-path problem. See [DESIGN_ISSUES.md](DESIGN_ISSUES.md#4-deerflow-gateway-config-issue).

### Hermes Orchestrator

| Property | Value |
|----------|-------|
| Port | None exposed |
| Image | ubuntu:22.04 |
| Version | v0.6.0 |
| Profiles | 9 (3×3 matrix) |
| Default | tiferet-beta |
| Status | Active |

**Role:** Memory and learning orchestrator. Manages the cognitive architecture via 9 switchable profiles organized in the Hermes 3×3 matrix.

**Profile switching:**
```bash
hermes -p kether-alpha chat    # High-level generation
hermes -p tiferet-beta chat    # Default: balanced orchestration
hermes -p malkuth-gamma chat   # Ground-level execution
```

See [ARCHITECTURE.md](ARCHITECTURE.md#hermes-3×3-harness-matrix) for the full matrix.

### Milvus Vector Database

| Property | Value |
|----------|-------|
| Port | 19530 (gRPC) |
| Image | milvusdb/milvus |
| Status | Active |

**Role:** Stores vector embeddings generated by nomic-embed-text for semantic search across the vault and conversation history.

**Future:** May be replaced by FAISS for lower memory footprint.

**Health Check:**
```bash
# gRPC health (requires grpcurl)
grpcurl -plaintext localhost:19530 milvus.proto.milvus.MilvusService/GetVersion
# Or simply test TCP
nc -z localhost 19530
```
